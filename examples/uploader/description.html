This is the <imgur-uploader-form> component. If you just want to drop a form onto a page that is capable of uploading an image with a description, this is the easiest way to do it.

Options
  • formClass (String): set a class on the form element
  • inputClass (String): set a class on the input elements
  • inputContainerClass (String): set a class on the inputs' container elements
  • submitClass (String): set a class on the submit button
  • descriptionLabel (String): label for description input (default: "Description")
  • imageLabel (String): label for file input (default: "Image")
  • submitLabel (String): label for submit button (default: "Upload")
  • onSubmit (Function): called when form is submitted, takes an uploadPromise argument which resolves to the response of the submission request
