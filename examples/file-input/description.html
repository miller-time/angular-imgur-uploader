This is the <imgur-uploader-file-input> component. If you just want to add a file input (that has imgur uploading capabilities) to an existing page or form, then this is the way to go.

Options
  • inputClass (@): set a class on the input element
  • uploaderConfig (Object): A scope Object with an onReady method that takes an uploaderApi argument. The uploaderApi can then be used to interact with the file that is selected. The uploaderApi has the following methods:
    • getFile: return file info about the selected file
    • submit: upload the selected file to imgur
